# OpenSSL-Tutorial

OpenSSLライブラリを使用したAES-256-CBC暗号化・復号化プログラムです。

## ソフトの内容

このプログラムは、OpenSSLライブラリのEVP APIを使用してAES-256-CBC方式の暗号化と復号化を実装しています。C++20で作成されており、コマンドラインから以下の機能を利用できます：

- **暗号化**: 標準入力から受け取ったデータをAES-256-CBCで暗号化して標準出力に出力
- **復号化**: 標準入力から受け取った暗号化データをAES-256-CBCで復号化して標準出力に出力
- **IV（初期化ベクトル）指定**: `-i`オプションで128ビット（16バイト）のIVを16進数文字列で指定
- **暗号化キー指定**: `-k`オプションで256ビット（32バイト）の暗号化キーを16進数文字列で指定

## コンパイル方法

### 必要なもの

- g++（C++20対応版）
- OpenSSLライブラリと開発ヘッダ
- make

### Ubuntu/Debianの場合

```bash
sudo apt-get install build-essential libssl-dev pkg-config
```

### コンパイル手順

```bash
make
```

このコマンドで`app`という実行ファイルが生成されます。

その他のmakeコマンド：

```bash
make clean   # ビルド成果物を削除
make re      # クリーンしてから再ビルド
```

## 使用方法

```bash
./app -i <IV（16進数）> -k <KEY（16進数）> <enc|dec>
```

例：

```bash
# 暗号化コマンドと復号化コマンドの互換性テスト
data="Hello World"
iv="00000000000000000000000000000001"
key="0102030405060708090a0b0c0d0e0f101112131415161718191a1b1c1d1e1f20"

# 暗号化して base64 で出力
echo -n "$data" | ./app -i "$iv" -k "$key" enc | base64

# base64 をデコードして復号化
echo '<BASE64_OUTPUT>' | base64 -d | ./app -i "$iv" -k "$key" dec
```

実際の運用では、ランダムなIVとキーを使用してください：

```bash
# ランダムなIVとキーでテスト
iv=$(head -c 16 /dev/urandom | od -x -A n | tr -d ' \n')
key=$(head -c 32 /dev/urandom | od -x -A n | tr -d ' \n')
echo "Test data" | ./app -i "$iv" -k "$key" enc | ./app -i "$iv" -k "$key" dec
```

## テスト用シェルスクリプトの処理内容

`test.sh`はこのプログラムがOpenSSL標準コマンドと互換性があることを検証するテストスクリプトです。

### 処理手順

1. **/dev/urandomから16バイトのランダムデータを生成し、16進数文字列に変換してIVとする**

2. **/dev/urandomから32バイトのランダムデータを生成し、16進数文字列に変換して暗号化キーとする**

3. **/dev/urandomから1000バイトのランダムデータを生成し、base64エンコードしてテスト用データとする**

4. **以下の3つのテストケースを実行して結果を表示**

   - **テスト1**: 自作プログラムで暗号化 → `openssl aes-256-cbc`コマンドで復号化
   - **テスト2**: `openssl aes-256-cbc`コマンドで暗号化 → 自作プログラムで復号化
   - **テスト3**: 自作プログラムで暗号化 → 自作プログラムで復号化

各テストケースで、復号化されたデータが元のテストデータと一致すれば、暗号化・復号化が正常に機能していることが確認できます。

### テスト実行方法

```bash
make              # まずコンパイル
./test.sh         # テスト実行
```
